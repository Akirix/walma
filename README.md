
# Collaborative whiteboard


# Installing

Install Node.js and npm.

    git clone git@github.com:opinsys/puavo-whiteboard.git
    cd puavo-whiteboard

Install dependencies

    npm install --dev

Develop run

    bin/develop

Production execute

    npm start
